# 🐍 Snake Game

A classic **Snake Game** built with **Python Turtle Graphics** and **Object-Oriented Programming (OOP)**. Control the snake, eat food to grow longer, avoid collisions, and try to achieve the highest score.

## ✨ Features

* 🎮 Keyboard controls (W, A, S, D or Arrow Keys)
* 🍎 Food collision detection
* 🐍 Snake growth animation
* 🏆 Real-time score tracking
* 💥 Wall collision game over
* 🔁 Self-collision detection
* ⚡ Smooth Turtle Graphics gameplay

## 🛠️ Built With

* Python 3
* Turtle Graphics
* Object-Oriented Programming (OOP)
* Random Module
